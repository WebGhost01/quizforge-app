# QuizForge + RecipeBox

Two apps: interactive quiz platform and recipe manager.

## QuizForge
- Custom quizzes with multiple choice questions
- Timed sessions with scoring & history

## RecipeBox
- Recipe storage with ingredients & steps
- Category filtering and search

## Stack
React 19 · TypeScript · Tailwind CSS · Vite · React Query · Shadcn/UI

## Run
```bash
npm install && npm run dev
```
Requires [portfolio-api-server](https://github.com/WebGhost01/portfolio-api-server).
